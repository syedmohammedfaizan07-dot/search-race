# Search Race 🚀

## 📌 Project Overview
This project compares the time complexity and execution time of different algorithms and data structures using a simple frontend UI.

## 🎯 Features
- Compare Bubble Sort vs Merge Sort
- Compare List vs Set membership check
- Measure execution time using Python `time` module
- Display results on frontend

## 🛠️ Tech Stack
- Frontend: React
- Backend: Python (Flask)
- Styling: CSS / Tailwind

## ⚙️ How It Works
1. User enters a sequence of numbers
2. Data is sent to backend
3. Algorithms run and execution time is measured
4. Results are displayed on UI

## 📊 Algorithms Used
- Bubble Sort → O(n²)
- Merge Sort → O(n log n)
- List Search → O(n)
- Set Search → O(1)
